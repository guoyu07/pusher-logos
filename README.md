# Logos

If you should wish to use our logos, do [download a zip file](https://github.com/pusher/pusher-logos/archive/master.zip) and take your pick.

## Usage
Feel free to use our logo but please don't change it.
When using our logo with other elements please make sure you leave adequate spacing. Don’t add elements, change the colour, distort our logo or use old versions.

Likewise, please don't use them as part of your own product, business, or service’s name.
